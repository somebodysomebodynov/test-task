Репозиторий:

Микросервис на Java с REST эндпоинтом, принимающем на вход одно число и возвращающий массив
отсортированных чисел.

Инструкции:

Запустить файл docker-compose. Все :) Для тестирования посылать post запросы на адрес 
http://localhost:8080/numbers. В качестве body передаем json, например такой: 

{
"value" : 1
}

Дорогой пользователь, желаем приятного использования этого чуда программистской мысли!
